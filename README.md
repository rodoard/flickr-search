# Flickr Search App

Searches flicker for tags in search field on 'Go' and renders a grid of photos with the view count centered over the thumbnail.


## Dependencies

Uses [ParcelJS](https://parceljs.org/) for bundling

## Getting Started

### Installing
```
npm install
```
### Verify installation

```
npm test
```
### Start Application

```
npm run start
```
